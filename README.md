# Installation

### Clone the project
```
git clone https://github.com/owiesnama/oktama-task.git
```
### Cd to the project and Copy env file
```
cd ./oktama-task && cp .env.example .env
```
### Run the migrations
```
php artisan october:up
```
### Serve the project

```
php artisan serve
```
